# Release Notes 1.1.1

## 中文 (Chinese)

本版本为个人定制自用版，主要新增了目录导航面板、视图切换滚动保持、侧边栏默认折叠等功能，并优化了多处 UI 交互体验。

### 1. 目录导航面板 (TOC Panel)
* **新增目录面板**：支持解析 Markdown 标题并生成右侧目录大纲，方便长文档的快速跳转导航。
* **快捷控制与折叠**：
  * 支持单独或一键全部展开/折叠目录节点。
  * 将目录的开关按钮移至底部的状态栏，提高界面整洁度和易用性。
* **匹配优化**：引入官方 `github-slugger` 处理标题的锚点 ID，解决中英文及特殊字符标题的跳转失效问题。

### 2. 视图切换滚动位置保持 (Scroll Position Preservation)
* **无缝位置保持**：解决了切换「编辑」、「分栏」、「预览」模式时页面跳回顶部的问题。
* **高精度块级映射**：定位时不依赖简单的比例换算，而是利用 Markdown 块结构进行高精度的位置对应定位，切换前后阅读/编辑进度无缝衔接。
* **多路径处理**：处理了全部 6 种切换路径（包括进入或退出分栏时宽度缩放导致的内容重排），完美保持视觉连贯性。

### 3. 侧边栏与启动优化 (Sidebar & Startup Optimization)
* **默认折叠配置**：新增「启动时默认折叠侧边栏」选项。
* **视觉细节修复**：修复了首次打开程序时配置未加载导致的侧边栏闪烁（Flash）以及多余的过渡动画，使启动更加丝滑。

### 4. 界面布局与细节优化 (UI & Layout Tweaks)
* **布局挤压修复**：重构了设置面板中的部分开关组件为按钮样式，解决了表单元素聚焦时导致的布局异常挤压。
* **目录样式调整**：优化了 TOC 面板 of 宽度（220px）和文本自动换行对齐。

## English

This release is a customized version for personal use. It features a new Table of Contents (TOC) panel, precise scroll progress preservation across view modes, startup sidebar collapse option, and various UI/UX refinements.

### 1. Table of Contents (TOC) Panel
* **New Navigation Panel**: Parses Markdown headings and displays an outline panel on the right side for quick heading jumps in long notes.
* **Toggles & Controls**:
  * Added collapse and expand buttons for individual headings as well as a master toggle to collapse/expand all headings at once.
  * Moved the TOC toggle switch to the status bar at the bottom for better discoverability and a cleaner toolbar.
* **Anchors Match Fix**: Integrated the official `github-slugger` package for generating slug IDs to resolve navigation mismatch bugs for special characters and bilingual headings.

### 2. Scroll Position Preservation Across View Modes
* **Seamless Progress Retention**: Resolved the annoying bug where switching between "Edit", "Split", and "Preview" modes reset the scrollbar to the top of the page.
* **Precise Block Mapping**: Replaced basic scroll ratios with a high-precision markdown block-level mapper to accurately synchronize reading and writing progress.
* **Robust Multi-path Transitions**: Handled all 6 mode-switching transition paths seamlessly (including text-reflow compensation when layout widths change in Split view).

### 3. Sidebar & Startup Enhancements
* **Collapse Sidebar by Default**: Added a new configuration option to automatically collapse the sidebar on application startup.
* **Glitches Fixed**: Fixed the initial sidebar flash and skipped transition animations during the bootstrap load, making the startup experience smooth.

### 4. UI Layout & Detail Improvements
* **Layout Squeezing Fix**: Refactored the form switches in the settings panel into buttons to resolve layout squeezing issues when focused.
* **Outline Style Tweaks**: Increased TOC panel width to 220px and added elegant text-wrapping with flex-alignment support.
